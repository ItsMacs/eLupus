# eLupus
Werewolf on a MC Server

## Modules
### lupus-game
- Game plugin managing the eLupus gamemode. Runs on Paper 1.21.8 and support multiple games on the same server.

### lupus-com
- COM player manager with simulated player movement. Chat and reasoning powered by DeepSeek.

### lupus-chat
- Chat plugin managing automatic translation (powered by DeepL) with request optimization.

### lupus-framework
- Framework for all Lupus plugins
